> This repo contains **2 ML projects** built around intelligent resume analysis:

<table>
<tr>
<td align="center" width="50%">

### 🧠 Project 1: RAG Pipeline
**Simple RAG Pipeline**

A foundational ML pipeline that loads resumes, chunks documents, builds FAISS vector embeddings, and uses LLM-powered semantic analysis to evaluate resume-to-job fit.

`LangChain` `FAISS` `OpenAI` `RAG`

</td>
<td align="center" width="50%">

### 🚀 Project 2: AI Resume Suite
**Full-Featured Web Application**

Production-ready platform with **5 AI features**: ATS Scoring, Keyword Extraction, Gap Analysis, Resume Tailoring (4 styles), and Cover Letter Generation — with PDF/DOCX export.

`Streamlit` `FastAPI` `ReportLab` `Render` `Vercel`

</td>
</tr>
</table>

<details>
<summary><b>📋 Full Feature List (click to expand)</b></summary>
<br>

| Feature | Description | Tech |
|---------|-------------|------|
| 🎯 ATS Scoring | Score 0–100% with section breakdown, matched/missing keywords, formatting checks | LLM + JSON parsing |
| 🔑 Keyword Extraction | 7-category keyword analysis from job descriptions | LLM structured output |
| 📊 Gap Analysis | Semantic resume-JD comparison with source chunks | RAG + FAISS + RetrievalQA |
| ✏️ Resume Tailoring | AI rewrites in 4 template styles, preserving facts | LLM + prompt engineering |
| 💌 Cover Letter | Personalized 3–4 paragraph letters | LLM generation |
| 📥 Export | Download as PDF or DOCX | ReportLab + python-docx |
| 🌐 REST API | 6 FastAPI endpoints for programmatic access | FastAPI + Uvicorn |
